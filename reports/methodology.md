# global-data-careers-intelligence-platform
A job market intelligence platform analyzing global data career demand, skills, salaries, remote opportunities, and career trends.

## Project Positioning
The Global Data Careers Intelligence Platform is a labor-market analytics project designed to help entry-level and junior data professionals make better career decisions using live job-market data. The project focuses on global and remote data-related roles, including Data Analyst, BI Analyst, Data Engineer, Analytics Engineer, and similar roles.
The core problem this project addresses is that job seekers often apply for roles without clear evidence of which countries, skills, tools, industries, or seniority levels offer the best opportunities. Job information is spread across multiple platforms, salary fields are inconsistent, skill requirements are hidden inside job descriptions, and remote hiring patterns are difficult to compare manually.
This platform solves that problem by collecting job postings from multiple APIs, storing structured job data, extracting skill requirements, analyzing salary and geography patterns, and producing dashboards that support career decisions. Instead of only showing job counts, the project is designed to answer practical questions such as where an entry-level candidate should apply, which skills are most demanded, which tools are linked to higher salaries, and what skill should be learned next.
The project is positioned as a career intelligence system rather than a simple job-posting dashboard. Its value comes from turning scattered labor-market data into decision-ready insights. The analysis is expected to reduce wasted application time, improve role targeting, identify stronger markets, and support a more strategic learning roadmap for data career growth.

### Decision Value

This project is built around one central analyst question:

If I hold this information, what decision improves?
The platform should improve decisions in four areas. First, it should help users identify countries with stronger remote data-role availability. Second, it should show which technical and business skills appear most often in real job postings. Third, it should connect skills, roles, and salary patterns to reveal which capabilities may increase earning potential. Fourth, it should guide entry-level candidates toward realistic next steps instead of generic career advice.

### Risk, Money, and Time Impact
The risk of not having this information is poor job targeting. A candidate may apply to roles, countries, or industries that offer limited entry-level opportunity. The money impact is that candidates may prioritize lower-paying markets or miss skills associated with stronger salary outcomes. The time impact is wasted effort, since weeks of applications may be spent on roles that do not match the candidate’s current skill level or target career path.
The project scope is limited to global and remote data-related roles. It does not attempt to analyze every profession. This controlled scope keeps the analysis focused, measurable, and relevant to data career planning.

### What This Project Proves
This project demonstrates the ability to define a real analytical problem, collect live data through APIs, work with JSON, build an ETL pipeline, store data in DuckDB, analyze data using SQL and Python, automate refreshes, and communicate insights through Tableau dashboards and written recommendations.
The final product should show that the analyst can move from raw labor-market data to practical career intelligence.
